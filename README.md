# Colegio Monteverde - Sitio Web Oficial# Colegio Monteverde - Sitio Web



Sitio web institucional del Colegio Monteverde, institución educativa en Peñalolén, Santiago de Chile.Rediseño moderno y profesional del sitio web del Colegio Monteverde.



## 📋 Descripción## 🎨 Características del Diseño



Sitio web completo multi-página con diseño profesional inspirado en instituciones educativas de excelencia. Incluye 14 páginas institucionales, sistema de noticias dinámico, formulario de contacto y backend API.### Mejoras Implementadas:

- ✅ **Diseño Moderno y Limpio**: Interfaz profesional que genera buena primera impresión

## 🎨 Diseño- ✅ **Tipografía Profesional**: Uso de Google Fonts (Montserrat y Open Sans)

- ✅ **Colores Institucionales**: Paleta de colores burdeo/rojo del colegio correctamente aplicada

- **Paleta de colores**: Navy blue (#1a365d) como color principal- ✅ **Responsive Design**: Se adapta perfectamente a móviles, tablets y desktop

- **Tipografía**: Montserrat (headings) + Open Sans (body)- ✅ **Navegación Intuitiva**: Menú fijo con navegación suave entre secciones

- **Estilo**: Profesional, limpio, inspirado en [Alcántara-Alicante](https://www.alcantara-alicante.cl/)- ✅ **Animaciones Sutiles**: Efectos visuales que mejoran la experiencia

- **Responsive**: Diseño adaptativo para móviles, tablets y desktop- ✅ **Organización Clara**: Contenido bien estructurado y fácil de encontrar
